iROV

Internet Remotely Operated Underwater Submersible Vehicle, or iROV for short. Designed to be a low-cost, easy-to-use, and accessible underwater drone. The system is designed to be accessible to anyone who has internet access. You can use a keyboard, game controller, or touchscreen. I mainly worked on the backend code, and some frontend, as well as physical testing, assembly, and more. Frontend Code Backend Code CAD Designs
